# Generated GraphQL Types

This directory contains auto-generated TypeScript types from GraphQL schema and operations.

⚠️ **DO NOT EDIT FILES IN THIS DIRECTORY MANUALLY**

These files are automatically generated by GraphQL Code Generator.

## Regenerate Types

```bash
pnpm run codegen
```

## What's Generated

- `graphql.ts` - All TypeScript types, query/mutation types, and document nodes
- `gql.ts` - Tagged template literal helper for writing inline queries
- `index.ts` - Main exports
- `fragment-masking.ts` - Fragment masking utilities for type safety

## Usage

```typescript
import type { GetEventsQuery, GetEventsQueryVariables } from '@/generated/graphql';
```

See `src/examples/` for usage examples.
